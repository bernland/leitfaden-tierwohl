=== Leitfaden Tierwohl ===

== Description ==

Mit diesem Plug-in können Quiz erstellt werden, um das Wohlergehen von Tieren anhand von einfach am Tier zu erhebenden Parametern einzuschätzen. Die Quiz dienen der Selbstevaluierung der Beurteilungsskompetenz sowie der Verbesserung der persönlichen Fähigkeiten zur Beurteilung des Tierwohls.
